#### 3.2 数据趋势图 (折线图)
- **位置**: 仪表板页面
- **尺寸**: 8列宽度
- **数据**: 近7天新增用户和作品
- **配色**:
  - 用户线: `#465C88` (主色)
  - 作品线: `#FF7A30` (辅助色)
- **特性**:
  - 平滑曲线 (tension: 0.4)
  - 填充渐变
  - 响应式
  - 交互提示

```javascript
trendChart = new Chart(ctx, {
  type: 'line',
  data: {
    labels: labels,
    datasets: [
      {
        label: '新增用户',
        data: userData,
        borderColor: '#465C88',
        backgroundColor: 'rgba(70, 92, 136, 0.1)',
        tension: 0.4,
        fill: true
      },
      {
        label: '新增作品',
        data: workData,
        borderColor: '#FF7A30',
        backgroundColor: 'rgba(255, 122, 48, 0.1)',
        tension: 0.4,
        fill: true
      }
    ]
  }
});
```
